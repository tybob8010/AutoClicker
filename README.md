# AutoClicker

### Latest Version : `v.1.1`

windows用の連打ツールです。

通常の連打機能に加え、<ins>設定の時間になると自動的に終了する機能があります</ins>。

CookieClickerを連打ツールを用いてプレイしつつPCのCPUを労わるために自動終了機能を作りました。
